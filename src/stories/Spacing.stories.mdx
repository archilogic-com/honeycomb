# Spacing

## Usage

Our design system defines a **spacing scale** used for any spacing between elements on the page.
The spacers naming follows a popular convention where `1 = 0.25rem`.

You can use them as classes generated by Tailwind CSS.
By default the spacing scale is inherited by the `padding`, `margin`, `width`, `height`, `maxHeight`,
`gap`, `inset`, `space`, and `translate` core plugins.
More info in the [Tailwind docs](https://tailwindcss.com/docs/customizing-spacing).

_Each story is named in the sidebar after the relevant class name._
