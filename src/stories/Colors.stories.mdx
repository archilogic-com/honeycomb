# Colors

## Usage

You can use these colors as classes generated by Tailwind CSS. They are automatically shared by all color-driven utilities, like `textColor`, `backgroundColor`, `borderColor`, and more.

_Each story is named in the sidebar after the relevant class name._

Additionally, there are `transparent` and `current` color classes.
