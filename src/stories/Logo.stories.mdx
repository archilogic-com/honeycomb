## Using as a button or link

When the Archilogic logo is meant to be clickable, use `<a-logo-button>`.
If it should behave like a link, provide an `href` attribute.

```html
<a-logo-button href="/" />
<!-- OR -->
<a-logo-button @click="doSomething" />
```

## Using as an svg image

### Default

At the moment, the `default` variant comes in one size only. You do not need to specify any props for it.

```html
<a-logo />
```

<Canvas>
  <Story name="logo default">{Template.bind({})}</Story>
</Canvas>

### Mark

The `mark` variant comes in 3 sizes. Providing a size prop for it is required.

```html
<a-logo variant="mark" size="md" />
```

### Color

The logo's default color is our [New York color](?path=/docs/primitives-colors--newyork).

For dark backgrounds, it's possible to change the color to `white`:

```html
<a-logo color="light" />
```
