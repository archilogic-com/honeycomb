# Separator

A simple separator component, rendering a horizontal or vertical line to section content on the page.

## Usage

```html
<a-separator />
<ASeparator />
```

<ArgsTable of={ASeparator} />

## Separator orientation

The separator can be `horizontal` (default) or `vertical`.

```html
<!-- "horizontal" is the default and can be omitted -->
<a-separator orientation="horizontal" />
<a-separator orientation="vertical" />
```

## Styling the separator

The separator stretches the full width (horizontal) or height (vertical) of the parent container.
To create a gap, add some padding to the parent container, or margin to the separator as appropriate.

**Note**: The vertical separator is designed to be used inside flex containers with "flex-row" layout.

## Rendering a different HTML element

```html
<a-separator as="li" />
```

The horizontal separator renders an `<hr>` element, and vertical separator renders a `<div>` element.
To render `<a-separator>` as a different HTML element use the `as` prop.
A typical use case is grouping items within a list (`<ul>` or `<ol>` elements) where the only valid child is `<li>`.
