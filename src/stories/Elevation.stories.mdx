# Elevation

## Usage

Our design system defines **3 elevation levels** achieved with box shadows.

You can use them as classes generated by Tailwind CSS.

_Each story is named in the sidebar after the relevant class name._

To remove box shadow, use `shadow-none`.

<Canvas isColumn="true">
  <Story name="shadow-sm">
    {() => {
      return {
        template: `
          <div style="width: 160px; height: 40px" class="shadow-sm flex justify-center items-center rounded body-sm"> Small </div>
        `
      }
    }}
  </Story>
  <Story name="shadow-md">
    {() => {
      return {
        template: `
          <div  style="width: 224px; height: 80px" class="shadow-md flex justify-center items-center rounded body-md"> Medium </div>
        `
      }
    }}
  </Story>
  <Story name="shadow-lg">
    {() => {
      return {
        template: `
          <div style="width: 448px; height: 160px" class="shadow-lg flex justify-center items-center rounded body-lg"> Large </div>
        `
      }
    }}
  </Story>
</Canvas>
