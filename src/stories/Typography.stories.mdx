# Typography

## Usage

Our design system defines several typeface styles for headings and body text.

You can use them as classes generated by Tailwind CSS.

_Each story is named in the sidebar after the relevant class name._

## Font

Our primary font is [Inter](https://fonts.google.com/specimen/Inter) from Google Fonts.
We use 3 weights: 400, 500, and 600.

The font is loaded by our Tailwind preset.

## Headings

There are **6 heading levels** available.
These classes can be used on any HTML element,
so please use **semantic elements** and maintain **proper heading hierarchy** for each page!

## Body

There are **4 sizes** of body text, each in **3 weight** variants.

_Note_ for normal font weight (400), the weight suffix can be omitted:
e.g. `body-lg` is the same as `body-lg-400`.
