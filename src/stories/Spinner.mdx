# Spinner

```html
<a-spinner />
<ASpinner />
```

<ArgTypes of={SpinnerStories} />

**This component renders an `svg` element with CSS animations applied. It doesn't include any wrapper elements or positioning styles, nor page overlays.**

## Default

The default spinner has color `warsaw` from our [dark neutrals palette](http://localhost:6006/?path=/docs/primitives-colors--warsaw#dark-neutrals).

```html
<a-spinner size="sm" />
<a-spinner size="md" />
<a-spinner size="lg" />
```
